Personal UPI Usage and Financial Analyzer using
Personal UPI Usage & Financial Analyzer using LLMs is an AI-powered FinTech application that analyzes UPI transaction data from platforms like Paytm, Google Pay, and PhonePe. The system processes transaction statements, generates spending insights, identifies wasteful expenses, visualizes trends, and provides personalized financial recommendations through an interactive Streamlit dashboard.

This project combines NLP, data analytics, and LLM-driven insights to enhance personal financial awareness and simplify multi-app UPI data analysis.
